## 1.0.0

* Initial release
* CustomLoader widget with 5 different animation styles:
  - Gradient: Rotating gradient effect
  - Dotted Circle: Animated dots forming a circle
  - Dual Ring: Two counter-rotating arcs
  - Wave: Pulsing waves radiating outward
  - Spinning Arcs: Multiple arcs spinning together
* Support for asset-based logos (includes built-in icon.png)
* Support for custom widget logos
* FullScreenLoader overlay component
* Customizable size, color, and stroke width
* Pulsing animation for center logo
* Common logo across all animation styles
